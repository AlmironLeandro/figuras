# Figuras


Se solicita modelar un conjunto de figuras geométricas a las cuales se les puede solicitar:

* superficie() : Double
* perimetro() : Double

## Figuras a Modelar:

### Circulo
Tiene como atributo `radio` de tipo `Integer`. Se calculan:

* superficie: Es Pi * Radio ^ 2
* perimetro() : Es 2 * Pi * Radio
